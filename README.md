# 鱼吃豆豆游戏 (Fish Eat Beans)

一个基于 HTML5 Canvas 开发的简单 2D 小游戏。

## 游戏玩法
- **控制小鱼**：使用鼠标移动或键盘方向键控制小鱼。
- **目标**：吃掉五颜六色的“豆豆”来增加分数。
- **规则**：
  - 玩家有 3 次生命。
  - 碰到画布边缘或红色障碍物会扣除一次生命。
  - 生命值为 0 时，游戏结束。

## 技术栈
- HTML5 Canvas
- 原生 JavaScript
- CSS3
